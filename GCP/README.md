# GCP (Garrysmod Communication Protocol)

​	GCP Required Pins :

​		From Adress Pins : [ A , B ]

​			[ A ]	Max : 255.255	

​			[ A ]	Min : 0.0

​			[ B ]	Max : 255.255	

​			[ B ]	Min : 0.0

​		To Adress Pins : [ C , F ]	

​			[ C ]	Max : 255.255	

​			[ C ]	Min : 0.0

​			[ F ]	Max : 255.255	

​			[ F ]	Min : 0.0

​		Send Pin : [ E ]	

​			[ E ]	Max : 1

​			[ E ]	Min : 0

​		Data Pins  : [ D , G , H ] 

​			[ D ]	Max : 255	

​			[ D ]	Min : 0

​			[ G ]	Max : 255

​			[ G ]	Min : 0	

​			[ H ]	Max : 255

​			[ H ]	Min : 0
